芬顿反应

实验原理：

亚铁离子通过单电子转移降低过氧键断裂所需的能量从而催化过氧化氢生成强氧化性的羟基自由基(HO•)和氢过氧自由基(HOO•) ，偏酸环境下该链式反应产生的自由基可以将反应体系中的其他有机化合物氧化为无机态

步骤1：Fe2+ + H2O2 → Fe3+ + HO· + OH-

步骤2：Fe3+ + H2O2 → Fe2+ + HOO· + H+

总反应：2H2O2 → H2O + HOO· + HO·

pH对反应的影响
pH ≤ 2.5 2.5 < pH < 3.5 pH ≥ 3.5
高浓度氢离子会淬灭•OH同时与铁离子形成稳定的络合物[Fe(H2O)6]2+阻碍反应进行 最佳反应条件 形成Fe(OH)2和/或Fe(OH)3沉淀，反应速率降低

实验试剂：

硫酸亚铁，3% (m/m) 过氧化氢，甘氨酸，氧化钙，2% (m/m) 稀硫酸

实验流程:

1. 配置澄清石灰水, 将0.5克氧化钙加入到200ml水中，静置24小时

2. 取一只试管作为检测管，缓慢吸取50ml静置后的上清液至检测管中

2. 另外取一只试管作为反应管，加入10ml 3%过氧化氢

3. 在一段橡胶管两端分别连接一段玻璃导管，其中一个玻璃导管插入到带孔橡胶塞中

4. 配置反应混合液，在20ml水中加入0.5克硫酸亚铁和0.1克甘氨酸，搅拌溶解后缓慢滴加稀硫酸将PH调至3

5. 将反应混合液加入含过氧化氢的反应管中，轻轻震荡试管使其混合均匀后盖上有玻璃导管的橡胶塞

6. 将橡胶管另一端的玻璃导管浸入到检测管的澄清石灰水中

7. 观察澄清石灰水是否变浑浊

实验结果

反应管由透明变为棕红色并产生大量气泡，亚铁离子被迅速氧化为棕红色的三价铁离子；十分钟后检测管中澄清石灰水变浑，芬顿反应生成的具有极强氧化性的自由基将反应体系中的唯一碳源甘氨酸(C2H5NO2)转为了无机物CO₂，二氧化碳与澄清石灰水中溶解的氢氧化钙反应生成不溶于水的碳酸钙
